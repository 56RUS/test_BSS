В папке _files находятся следующие данные:
  * data - папка с xml-файлами, которые прилагались к заданию
  * bss.exe - исполняемый файл программы
  * db_with_data.sql - скрипт создания БД с таблицами, заполненными данными.
    Данные были добавлены в БД программой bss.exe после обработки xml-файлов в папке data
  * empty_db.sql - скрипт создания пустой БД с пустыми таблицами
  * result_work_program.xml - результат выгрузки программой bss.exe данных из БД.
    Данные в БД были сохранены этой же программой после обработки xml-файлов в папке data
	
	
Из xml-файлов в БД сохраняются только те данные, у которых есть минимум один валидный CalcData.
Если у OwnerData список CalcsData пустой или содержит невалидные данные ((Param1 + Param2) != CheckSum), то такой OwnerData в БД добавлен НЕ БУДЕТ!





---------------------------------------- Задание -----------------------------------------------------------
В архиве содержатся несколько файлов в xml формате.
Описание структуры файлов:
    - OwnerData
        - OwnerName - имя
        - OwnerDate - дата просчетов
        - CalcsData - массив просчетов
            - Param1 - параметр для просчета
            - Param2 - параметр для просчета
            - CheckSum - сумма параметров для просчета

1) Сделать программу для получения данных из всех файлов в архиве.
2) Модифицировать программу так, чтобы была возможность проверки корректности данных в CalcData.
3) Сформировать БД для хранения данных, описанных в xml файлах.
4) Модифицировать программу так, чтобы данные из xml файлов, которые прошли проверку на корректность, были занесены в созданную БД.
5) Сделать в программе возможность получения данных из БД и сохранения в файл описанного формата.

По возможности, работу над проектом вести с использованием локального GIT.
Результатом должен являться архив папки проекта и копия БД или ее схема и выгрузка полученных данных.
